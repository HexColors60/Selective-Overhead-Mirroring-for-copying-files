Περιεχόμενα Φακέλου:
README

Makefile: εντολές make / make clean.

tools.c/tools.h: Διάφορες συναρτήσεις οι οποίες χρησιμοποιούνται σε διάφορα τμήματα κώδικα. Περιέχονται και οι συναρτήσεις read_data() write_data() και send_data(), οι οποίες είναι σχεδόν ίδιες με τον κώδικα που δίνει για αποστολή και λήψη δεδομένων ο κ. Δελής στο κεφάλαιο 6.

DataStructures.c/DataStructures.h: Ορισμοί και συναρτήσεις για τα structs που χρειάζονται τα προγράμματα.

MirrorInitiator.c: Διαβάζει από τα ορίσματα αυτά που χρειάζεται, στέλνει το κατάλληλο αίτημα στον 
MirrorServer και περιμένει την απάντηση του.

Worker.c: Οι συναρτήσεις για πρόσβαση/έξοδο του worker στον buffer, καθώς και η υλοποίηση της ρουτίνας worker που δημιουργεί ο MirrorServer.

MirrorManager.c: Οι συναρτήσεις για πρόσβαση/έξοδο του MirrorManager στον buffer, καθώς και η υλοποίηση της ρουτίνας MirrorManager που δημιουργεί ο MirrorServer.

MirrorServer.c/MirrorServer.h: Η Main του προγράμματος MirrorServer, όπου δέχεται ένα αίτημα, φτιάχνει τους αντίστοιχους Worker και MirrorManager, το εκτελεί και στέλνει τα αποτελέσμα του στον MirrorInitiator.

ReaderWriter.c: Υλοποίηση συναρτήσεων για συγχρονισμό της μορφής Readers - Writers, η οποία χρησιμοποιείται για να έχουν πρόσβαση τα services του Content Server στον πίνακα των delayID.

ContentServer.h: Ορισμός του struct delayID.

ContentServer.c: Η main του προγράμματος ContentServer, καθώς και η ρουτίνα ContentChild η οποία χρησιμοποιείται για την απάντηση οποιουδήποτε νέου αιτήματος (LIST/FETCH).

Αναφορές κώδικα:
Οι συναρτήσεις read_data(), write_data(), send_data() είναι μικρές παραλλαγές των συναρτήσεων read_data() write_data() του κώδικα του κ. Δελή στο κεφάλαιο 6. Επίσης, η σύνδεση των προγραμμάτων είναι από το ίδιο κεφάλαιο. Τέλος, η συνάρτηση removeSubstring στο tools.c δεν είναι δική μου αλλά την βρήκα στο https://goo.gl/ct99NZ και την χρησιμοποιήσα θεωρώντας ότι δεν υπάρχει πρόβλημα αφού δεν είναι το ζητούμενο της άσκησης.
